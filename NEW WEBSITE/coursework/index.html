Information about the practical components of IGCSE Music: Performing and Composing.

üé§ Component 2: Performing

Record two performances:

  • Solo Performance - Playing or singing by yourself (with accompaniment if needed)
  • Ensemble Performance - Performing with other musicians

Each should be at least 2 minutes. Total maximum 10 minutes.

üéµ Component 3: Composing

Create two original compositions:

  • Composition 1 - Free composition (your choice of style)
  • Composition 2 - Set brief (responding to a given stimulus)

Each should be 1-3 minutes. Include written commentaries.

Marking Criteria

Component Weight Key Criteria
Component 2: Performing 30% Technical control, expression, accuracy
Component 3: Composing 30% Ideas, development, use of elements

Submission Tips

  • Record in a quiet environment with good audio quality
  • Practice thoroughly before recording
  • Include score/notation with compositions
  • Write clear commentaries explaining your creative choices
